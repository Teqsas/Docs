# 3 Product Description

## 3.1 Scope of delivery
The scope of delivery includes the following parts:

* 1x LAP-TEQ PLUS ELEVATION sensor
* 1x Safety rope
* 1x Manfrotto 5/8” sleeve (for mounting on the MOUNTING BRACKET)
* Instructions for use and safety data sheet

4. XLR socket for connection to the readout unit (LAP-TEQ PORT)

!!! info "Note"
    In the factory setting, the trigonometric correction is activated and the unit is set to "m" (metres). To deactivate the trigonometric correction and/or to change the unit to "ft" (feet), please proceed as described in the chapter "Trigonometric Correction / Length Unit" on page 6.

The LAP-TEQ PLUS ELEVATION module is intended for mounting on a flying frame (using the LAP-TEQ PLUS MOUNTING BRACKET) and measures the vertical distance between the zero line on the device (see chapter Your device at a glance on page 9) and the surface located below it. This provides information about the height of a loudspeaker element or of the entire array. The laser beam of the ELEVATION serves as visual feedback for the area up to which measurement is taken. This way you can see whether the end point of the measurement is correct. A LAP-TEQ PLUS Sensor can additionally be attached to the MOUNTING BRACKET. This allows simultaneous measurement of height and vertical alignment. Both values can be read on the display at the same time. If a LAP-TEQ MOTION System is used as the display unit, height and alignment can be adjusted simultaneously.
