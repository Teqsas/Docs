# 1 Before you begin

## 1.1 Intended use
The LAP-TEQ PLUS ELEVATION sensor is used to determine the correct mounting height of loudspeakers. The LAP-TEQ ELEVATION sensor measures the vertical distance between the device and the surface located directly below it. The measured distance is shown on a separately purchased LAP-TEQ PLUS Display, LAP-TEQ PLUS Interface or a LAP-TEQ Motion System. Any other use that is not described in these instructions for use is unauthorised misuse. The manufacturer is not liable for any damage resulting from this. Never use the device in public traffic (road traffic, aviation, etc.).
