# 3 Product

## 3.1 Your device at a glance

![LAP-TEQ PLUS INTERFACE](../../../assets/interface/lap-teq-interface-product.jpg){ width=480 }

The interface is a handy device that is powered by 230V. It is available as a standalone version with three sensor inputs and as a plug-in board for the LAP-TEQ motion receiver. The three sensor inputs are designed as LAP-TEQ interfaces and can be connected to all LAP-TEQ sensors, such as the ELEVATION, the ATMOSPHERE or the INCLINOMETER, among others.

The measured values fed in this way are forwarded to a TCP/IP interface (RJ45) and can thus be read via a web interface on your end device.

In addition, in order to read out an even larger number of sensors at once via your WEB interface, several web displays can also be integrated into a network at the same time.

You can also find more information on this under the following link:
[https://www.youtube.com/watch?v=xWBK6l7K8Mc](https://www.youtube.com/watch?v=xWBK6l7K8Mc)
