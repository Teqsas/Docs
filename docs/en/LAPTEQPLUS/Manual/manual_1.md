# 1 Before you start

## Intended use
The LAP-TEQ PLUS is used to measure the inclination of loudspeaker systems.

The LAP-TEQ PLUS measures the vertical angle between its longitudinal axis and the surface of the earth. This provides information about the vertical alignment of the loudspeaker element.

Any other use that is not described in these instructions for use is unauthorised misuse. The manufacturer is not liable for any damage resulting from this.

Never use the device as a display device or in public traffic (road traffic, aviation, etc.).
