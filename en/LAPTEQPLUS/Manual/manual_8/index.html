8 Calibration

Note

The INCLINOMETER is calibrated at the factory for operation with the base plate facing down.

If you wish to operate the INCLINOMETER with another orientation of the base plate (up, left or right), calibration in the desired position is required.

Once all positions have been fully calibrated, the INCLINOMETER can be used in all positions without further calibration.

A new calibration may become necessary if the INCLINOMETER has been exposed to extreme shocks (e.g. from being dropped). Calibration is required if the INCLINOMETER is not to be operated with the base plate facing down — i.e. with the base plate facing up, left or right. The calibration procedure is identical for all positions except for the orientation of the base plate, and is therefore described in this manual using the variant "base plate facing down" as an example.

In addition to the INCLINOMETER to be calibrated, the following materials are required:

  • Display module
  • Calibration cable (see image in Chapter 3.2). Standard XLR cables are not suitable.
  • Spirit level
  • Set square

Step 1

  • Connect the INCLINOMETER to the display module using the calibration cable.

Step 2

  • Check that the reference surface on which calibration is to be carried out is level.
  • Position the INCLINOMETER with the base plate on the reference surface. The laser outlet faces left.

Step 3

  • Press and hold the "CAL button" on the display module.
  • Press the "Power button" on the display module.
  • Once the "Welcome" text appears at the bottom of the display, release both buttons.

The system is now in calibration mode and indicates this in the upper area of the display by showing "start calibrte?".

Note

If the "CAL button" on the display module is pressed during operation, the INCLINOMETER is not put into calibration mode. Instead, brief calibration instructions appear on the display. After a few seconds, the display automatically returns to the angle reading.

  • Press the "CAL button" again to start the actual calibration.

The display shows the question "calibrte 0.0° A?".

  • Press the "CAL button" again to start calibration in the "A" direction.

If the display shows "Sens pos incorrect", please check the orientation of the base plate.

If the procedure is correct, the display shows "don't move" – followed by a numerical value for "A" and the question "calibrte 0.0° B?".

Step 4

  • Rotate the INCLINOMETER so that the laser outlet faces right.
  • Press the "CAL button" again to start calibration in the "B" direction.

The display again shows "don't move" – followed by a numerical value for "B" and the question "calibrte +45.0°?".

Step 5

  • Use a set square to align the INCLINOMETER so that the laser shines upwards at a 45° angle.
  • Press the "CAL button" again to start calibration at "+45.0°".

The display shows "don't move" – followed by a numerical value for "+45.0°" and the question "calibrte −45.0°?".

Step 6

  • Use a set square to align the INCLINOMETER so that the laser shines downwards at a 45° angle.
  • Press the "CAL button" to start calibration at "−45.0°".

The display shows "don't move" – followed by a numerical value for "−45.0°".

Calibration for the orientation with the base plate facing down is now complete. The display shows "Calb Data stored".

Step 7

If the INCLINOMETER is operated exclusively with the base plate facing down, nothing further needs to be done.

If operation with another orientation is desired, the steps from Step 8 onwards must be performed.

To leave calibration mode, switch off the display module by pressing the "Power button". The next time it is switched on – without pressing the "CAL button" – the device returns to its normal operating mode.

Step 8

If you want to operate the INCLINOMETER with the base plate in another position, calibration must be continued.

The order of the four possible positions is fixed as follows (viewed from the XLR connector of the INCLINOMETER):

Orientation Display text
I. Base plate down (factory calibration) Mpl Down¹
II. Base plate up Mpl Up
III. Base plate left Mpl Left
IV. Base plate right Mpl Right

¹ Mpl: Mounting plate.

Note

For better clarity as to which orientation (I.–IV.) is currently being calibrated, the display briefly shows the corresponding display text from the table.

Note

Note on Step 3: as the system is already in calibration mode, the "Power button" does not need to be pressed. Calibration starts immediately with calibration of position "A" when the "CAL button" is pressed.

If Step 2 to Step 6 have been performed correctly with the changed orientation of the base plate without the display showing "Sens pos incorrect" in between, calibration of this orientation finishes with the display showing "Calb Data stored".

The INCLINOMETER can now be operated with the base plate facing down and the base plate facing up. If these positions are sufficient, calibration mode can be exited by switching off using the "Power button".

If the lateral positions are also desired, Step 2 to Step 6 must first be repeated with the base plate facing left (viewed from the XLR connector of the INCLINOMETER). For the orientation with the base plate facing right (viewed from the XLR connector of the INCLINOMETER), calibration Step 2 to Step 6 must be performed once more.

The INCLINOMETER can now be operated in all four positions.

8.1 Calibration of 2 axes

  • Press and hold the "CAL button" on the display module.
  • Press the "Power button" on the display module.
  • Once the "Welcome" text appears at the bottom of the display, release both buttons.

The system is now in calibration mode and indicates this in the upper area of the display by showing "start calibrte?".

  • Press and hold the "CAL button" on the display module for approx. 20 seconds to enter "2-axis mode".

The display shows the question "calibrte 0.0° A?".

  • For further calibration, follow the description as outlined in Step 3, point 5 up to and including Step 6.

In addition, calibration of the X-axis at +45°/−45° follows.

After completing calibration, the X- and Y-axis values appear on the display. After switching off and back on, the "spirit level" view is shown.
